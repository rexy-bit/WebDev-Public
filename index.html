Home Page

This Employee Management Web Application is a full-stack project designed to help organizations efficiently manage their workforce data. Built with HTML, CSS, and JavaScript for the front-end, and powered by a Node.js server for the back-end, it allows users to perform key operations such as adding new employees, viewing the full list of employees, deleting or updating employee information, and searching for employees by their unique code. The application also provides real-time statistics, including the total number of employees in each department, helping users better analyze staff distribution across the organization. All data is securely stored in a local JSON file on the server, ensuring data persistence even after the browser is closed or the page is refreshed. The project focuses on simplicity, speed, and user-friendly interaction, making it a practical solution for small to medium-sized businesses.
