Home page

This project is a Logistics Delivery Management System developed as a web-based application using HTML, CSS, and JavaScript. It simulates a backend API by leveraging JSON Server, allowing CRUD operations (Create, Read, Update, Delete) on delivery records stored in a local JSON file. The system enables users to view, add, edit, and delete deliveries, with key details such as delivery type (incoming/outgoing), supplier, product, quantity, and status. This project helped me practice working with RESTful APIs, handling HTTP requests (GET, POST, PUT, DELETE), and connecting a Frontend to a local API environment without the need for a full backend server.
