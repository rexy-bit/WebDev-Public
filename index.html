The Stock Management Application is a user-friendly and responsive web-based system designed to simplify inventory management processes. Developed entirely with HTML, CSS, and JavaScript, this application allows users to efficiently handle their stock with a variety of essential features. Users can add new products by entering key details such as name, quantity, and price, as well as delete products that are no longer needed. The product listing feature displays all items in a clear and organized layout, making it easy to monitor available stock. The application also offers a search functionality to quickly locate specific products, along with a modification option to update product information as needed. Additionally, a dedicated statistics module provides valuable insights into total stock levels and overall inventory value. The sorting feature helps users arrange products by name, quantity, or price for better visibility and decision-making. To ensure a seamless experience across all devices, the application includes a responsive design with a hamburger menu that enhances navigation on mobile screens. This project highlights core front-end development skills such as DOM manipulation, responsive UI design, and client-side data storage using LocalStorage.
