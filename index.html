Booky is a feature-rich online bookstore built entirely with HTML, CSS, and vanilla JavaScript, designed to offer users a smooth and interactive experience while exploring books. The platform allows visitors to browse a visually appealing list of books, each presented with a cover image, title, author, and price. Users can seamlessly add books to their shopping cart, review their selections, and update the cart in real time, all without page reloads. The site also includes a powerful search functionality that enables users to quickly find books by typing keywords related to the title. Built as a single-page application (SPA) using hash-based routing, Booky dynamically switches between different sections such as the homepage, book list, search page, and cart, offering a fluid navigation experience. This project demonstrates how core web technologies can be used to build a lightweight but effective e-commerce platform without relying on external libraries or frameworks, making it an excellent foundation for anyone learning frontend development.
