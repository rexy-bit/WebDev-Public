Home Page

This project is an employee management system developed using HTML, CSS, and JavaScript. It allows users to efficiently manage a database of employees directly within the browser. The system includes key features such as adding new employees with their details (name, position, salary, etc.), deleting an employee based on a unique code, searching for a specific employee by their code, displaying the complete list of registered employees, and generating useful statistics including the average, minimum, and maximum salary. All data is stored locally using the browser's localStorage, making the application fully functional without the need for an external database.
