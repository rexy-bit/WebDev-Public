Home Page

This project is an Employee Management System developed using HTML, CSS, and JavaScript. It is designed to simulate real-world backend interactions by using a REST API powered by JSON Server. The employee data is stored in a local JSON file, acting as a mock database. Through this application, users can perform key operations such as displaying the list of employees (using GET requests), adding new employees (POST), deleting existing employees (DELETE), and soon, updating employee information (PATCH/PUT). This project helped me gain practical experience in working with APIs, understanding HTTP methods, handling JSON data, and building dynamic web applications that communicate with a backend service. It also strengthened my skills in frontend development, data management, and asynchronous JavaScript programming using the Fetch API.
