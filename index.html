Home Page

Welcome to the ABB Logistics Delivery Management System. This internal web application has been developed to help the ABB Logistics Department effectively manage all incoming and outgoing deliveries. The system offers a range of features including the ability to display the full list of deliveries, add new deliveries, delete existing ones, and update delivery details such as type, supplier, quantity, and status. It also includes a search function that allows users to quickly find a delivery based on its unique delivery code. Additionally, the application provides useful statistics on the total number of deliveries, their distribution by type (incoming or outgoing), and the current status of each delivery. This lightweight and user-friendly tool runs entirely in the browser using HTML, CSS, JavaScript, and localStorage, making it an efficient solution for daily logistics operations within ABB.
