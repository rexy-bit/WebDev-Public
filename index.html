Home Page

This IT Asset Management System is a web-based application designed to help companies like ABB effectively track and manage their hardware equipment. The system allows users to add new assets by entering details such as the equipment name, type, purchase date, status, and assigned employee or department. It also features a dynamic display of the full equipment list, making it easy to monitor and control the inventory. Users can search for specific equipment using its unique code, update equipment information like operational status or assignment, and delete equipment records when necessary. Additionally, the system provides useful statistics such as the total number of assets, the number of items in service, under maintenance, or out of service. The project is built using HTML, CSS, and JavaScript, and communicates with a local JSON-based API for data storage and real-time updates. This project reflects my growing skills in front-end development, API integration, and dynamic data management.
