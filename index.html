Home Page

This Student Management System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to efficiently manage a list of students through a user-friendly interface. The system includes key features such as adding new students, displaying the full list, searching by name, editing existing entries, viewing real-time statistics (like average score and total number of students), and sorting students based on their average grades. The application is structured across multiple HTML pages for better navigation and uses localStorage to ensure data is saved even after the browser is closed.
