This Web App is a Leave Management System with a Single Page Application (SPA) developed using HTML, CSS, and JavaScript for the RH departement of ABB. This internal tool allows employees to submit leave requests digitally, while administrators can view, approve, or reject them in real time. It includes a role-based login system, a dynamic leave request form, personal request tracking for employees, an admin dashboard for request management, and a statistics section to analyze leave data. All functionality is handled client-side using JavaScript and localStorage, with seamless navigation between views to ensure a smooth user experience.
