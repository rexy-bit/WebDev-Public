Home Page

A fully web-based Employee Training Tracker designed to streamline the management and monitoring of employee training sessions within a company. The system provides HR managers with a user-friendly interface to add detailed training records, including employee name, training title, date, and status (Ongoing or Completed). It offers a dynamic and responsive list view that displays all training sessions with the ability to sort by date or status for better organization. An integrated search functionality allows quick filtering of records based on employee names, making it easy to locate specific trainings. Users can also modify existing records to keep the database up to date. The application includes a real-time statistics section that calculates and displays useful insights, such as the total number of training sessions, the number of completed trainings, and overall completion rates. Built entirely with HTML, CSS, and JavaScript, the project leverages LocalStorage to ensure data persistence without the need for a backend server. This project focuses on front-end development best practices, including DOM manipulation, event-driven programming, data filtering, sorting algorithms, and client-side storage management, making it a practical solution for internal HR management needs.
